# Design System

## Colors
- Primary background: `--bg-primary`
- Panel background: `--bg-panel`
- Card background: `--bg-card`
- Accent primary: `--accent-primary`
- Accent secondary: `--accent-secondary`

## Typography
- Base font: `Inter`, `Segoe UI`, system UI
- Headings: 24-32px with 600-700 weight
- Body: 13-16px with 400-500 weight

## Spacing
- Base spacing unit: 8px
- Card padding: 16-24px
- Section gaps: 16-24px

## Components
- Cards: `card`, `card soft`
- Buttons: `primary`, `ghost`, `small`
- Metrics: `metric-card`, `metric-label`, `metric-value`
